# Tríptico GRIDBOT

Este directorio contiene una versión en código del tríptico informativo de GRIDBOT lista para abrir en navegador y exportar como PDF.

## Archivos

- `index.html`: estructura del tríptico
- `styles.css`: estilos de pantalla e impresión

## Tamaño

El diseño está preparado para:

- ancho: `27.9 cm`
- alto: `21.6 cm`

Cada hoja representa una cara completa del tríptico con 3 paneles:

- hoja 1: exterior
- hoja 2: interior

## Cómo usarlo

1. Abre `index.html` en tu navegador.
2. Revisa el contenido y el acomodo visual.
3. Pulsa `Exportar / Imprimir`.
4. Guarda como PDF o imprime en tamaño real.

## Recomendación

Al exportar:

- usa orientación horizontal
- conserva escala al `100%`
- desactiva encabezados y pies del navegador
- usa fondo gráfico habilitado si tu navegador lo pide
