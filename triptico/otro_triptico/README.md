# Triptico GRIDBOT · Version alternativa

Este directorio contiene una version alternativa del triptico de GRIDBOT con dos caras separadas y una vista unificada para exportar.

## Archivos

- `gridbot_triptico_cara_frontal.html`: cara frontal
- `gridbot_triptico_cara_trasera.html`: cara trasera
- `index.html`: visor unificado para abrir ambas caras desde una sola pagina
- `styles.css`: estilos del visor y configuracion de impresion

## Tamano

La vista esta preparada para:

- ancho: `27.9 cm`
- alto: `21.6 cm`

Cada iframe representa una cara completa del triptico.

## Como exportarlo

1. Abre `index.html` en tu navegador.
2. Revisa la cara frontal y la cara trasera.
3. Pulsa `Exportar / Imprimir`.
4. Guarda como PDF.

## Recomendaciones de impresion

- usa orientacion horizontal
- deja escala en `100%`
- desactiva encabezados y pies del navegador
- activa fondos graficos si tu navegador lo solicita

## Alternativa

Si prefieres exportar cada cara por separado, abre directamente:

- `gridbot_triptico_cara_frontal.html`
- `gridbot_triptico_cara_trasera.html`
